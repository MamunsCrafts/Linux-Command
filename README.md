# Linux-Command 

## Package Management

### Update Packages List
```
sudo apt update
```

### Upgrade Packages 
```
sudo apt upgrade
```

### Install debian packages

```
sudo dpkg -i file.deb
```


### Search Packages 

```
dpkg -l | grep virtualbox
```
### UnInstall debian packages

```
sudo apt-get remove --purge <package-name>

```
